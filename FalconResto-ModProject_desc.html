Overview

The Engineered-By-Aaron Falcon Resto-Mod Project is a custom data acquisition system integrated into a 1961 Ford Falcon, designed to extract and display real-time vehicle data via a Holley Terminator X Max ECU, including RPM, speed, odometer, oil temperature, coolant temperature, battery voltage, and air/fuel ratio.

System Architecture

The system architecture comprises the following key components:

  • Holley Terminator X Max ECU: The central processing unit responsible for data acquisition, processing, and display control.
  • CAN Bus Interface: A bidirectional communication system enabling data exchange between the ECU and the Arduino microcontroller.
  • Arduino Mega: A microcontroller used to process data from the CAN bus, control the display, and implement custom logic.
  • Miata Gauge Cluster: A custom-built gauge cluster fabricated in the template of a 1990 Mazda Miata, housing the five analog gauges and two digital screens.

Software Stack

  • Arduino
  • Embedded C
  • C++
  • LVGL (Graphic Library for Digital Gauge Design)
